gcommons.deutscher Installation
-------------------------------

To install gcommons.deutscher using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``gcommons.deutscher`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        gcommons.deutscher
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
